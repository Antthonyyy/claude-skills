Перед тем как генерировать и отправлять ответ любому контакту в Telegram/WhatsApp — обязательно прочитать полную историю переписки с ним из БД и напрямую из Telegram/WhatsApp.

**Why:** Без контекста предыдущих сообщений бот генерирует неподходящий ответ. Пример: Юлия Элайми ответила "Здравствуйте, давайте" на первое касание с приглашением на нетворкинг — нужно было отправить ссылку на регистрацию, а бот ответил общим "О чём вы хотите поговорить?".

**How to apply:** При любой ручной обработке входящего сообщения — сначала `get_messages(entity, limit=20)` из Telegram + запрос в `outreach.messages` по contact_id, потом генерировать ответ с полным history.
