**ДВА равноценных бесплатных онлайн-события (с 29.05.2026):**
- Нетворкинг — **среда 17:00 по Амстердаму**, Zoom.
- Бизнес-завтрак — **суббота 10:00 по Амстердаму**, Zoom. Раньше был закрытым перком пакета («не приглашай сама») — теперь второй бесплатный вход, бот зовёт сам.

**Маршрутизация (зови на БЛИЖАЙШЕЕ по календарю, точка переключения = час старта):** пн/вт/ср-до-17 → среда; ср-после-17/чт/пт/сб-до-10 → суббота; сб-после-10/вс → среда. Реализовано в `llm_tatiana._pick_nearest_event` + блок «БЛИЖАЙШЕЕ СОБЫТИЕ» в контексте времени + `{event_when}` в шаблонах рассылки (`next_event_invite_phrase`). Название «бизнес-завтрак» НЕ меняли (омоним с платными офлайн-ланчами). «Первое бесплатно» = одно знакомство суммарно (стадия `attended`). Деплой = рестарт listener'а. Связано: [[project_wnw_bot_prompt_architecture]].

**Why:** Перепутал дату — сказал Антону что нетворкинг "сегодня" (23.04), хотя он был вчера (22.04). Плюс 23.05 шаблоны рассылки (`clients/wnw/tasks/messages/v7,v8,v9,v10_bio.txt`) были зашиты на прошедшую «20 мая» — чуть не разослали приглашение на прошедшую дату 256 контактам.

**How to apply:** Всегда проверять день недели перед тем как говорить о дате нетворкинга. Если сегодня не среда — нетворкинг либо вчера, либо в ближайшую среду. **Перед каждой рассылкой проверять дату внутри шаблонов first-touch — она захардкожена и протухает каждую неделю.** Обновлять этот файл после каждой прошедшей встречи.
