Любой текст, написанный от имени Антона, должен пройти проверку на AI-паттерны.

**Why:** Антон хочет, чтобы контент звучал как живой человек, а не как ChatGPT. Клиенты и аудитория узнают AI-тексты.

**How to apply:** Перед финализацией любого текста (сообщение клиенту, пост, описание) — прогнать через этот чек-лист.

## Запрещённые фразы (RU/UA)

- "В современном мире" / "У сучасному світі" → начни с факта
- "Давайте разберёмся" → убрать, просто начни
- "Важно отметить" → убрать
- "На сегодняшний день" → убрать или назвать дату
- "Ключевой момент / аспект" → назови конкретно
- "Данный" → "этот"
- "Является" → "это"
- "Осуществлять" → "делать"
- "В рамках" → "в", "на", "для"
- "Оптимизировать" → "улучшить" или конкретно что
- "Масштабировать" → "вырасти", "увеличить"
- "Трансформация" → описать что было — что стало
- "Синергия" → убрать
- "Экосистема" (если не про природу) → "система"
- "Безусловно / Несомненно" → убрать
- "Хочу поделиться" → просто дели
- "Не секрет, что" → начни с факта

## Запрещённые структуры

- **Зеркальное начало** — AI пересказывает запрос первым предложением
- **Проповедь из трёх пунктов** — вступление → 3 пункта → вывод повторяющий всё
- **Резюме в конце** — "Итак, мы рассмотрели..."
- **"С одной стороны... с другой стороны"** — имей позицию
- **Вопрос → Ответ через строку** — "В чём секрет? В последовательности."
- **Нарастающий восторг** — "Первое... Второе... И самое важное!!!"

## Фейковые переходы — убить

- "При этом...", "Стоит отметить...", "Интересно, что..."
- "Но вот в чём дело...", "Тем не менее..."

## 7-Point Sniff Test (перед публикацией)

1. Это мог написать кто угодно? → Добавь свою историю/числа
2. Есть запрещённые фразы? → Найди и замени
3. Начинается с пересказа темы? → Перепиши начало
4. Заканчивается резюме? → Отрежь последний абзац
5. Каждое утверждение подкреплено числом? → Если нет — добавь или убери
6. Клиенты узнали бы это как ТВОЙ текст? → Сверься с Voice Profile
7. Ты бы это сказал вслух другу? → Прочитай вслух, перепиши что звучит как брошюра
