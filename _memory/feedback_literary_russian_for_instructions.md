Когда пишем инструкции, памятки, регламенты, документы — использовать только литературный русский.

**Why:** Антон раскритиковал черновик WhatsApp-памятки за жаргон («словить жалобы», «догоны», «дожимать базу», «копипастом», «Broadcast», «влез в группы») и несогласованный регистр (мешались «пиши»/«убираем»/«не отправляй»). Просил «подключить нормальный словарь» — то есть держать редакторский стандарт.

**How to apply:**
1. Литературный русский. Без сленга, продажного жаргона, англицизмов, бытовых метафор («номер живёт», «страховка от жалоб»).
2. Единый регистр во всём документе. Для инструкций по умолчанию — обращение на «вы» к читателю (выбор Антона 2026-05-19). Не смешивать с «мы»/«ты»/инфинитивом.
3. Термины по-русски; иностранный оригинал — в скобках один раз при первом упоминании.
4. Без разговорных подзаголовков («Суть в одной строке») и пустых связок («всё остальное — следствие этого»).
5. Деловой тон, но без канцелярита (запреты из [[user_anti_ai_guide]]: «осуществлять», «в рамках», «данный», «является»).

Отличие от [[user_voice_profile]]: тот профиль — для личных сообщений Антона (короткие, скобочки, разговорно). Это правило — для документов и инструкций, противоположный регистр.
